Brick Breaker Game (Java Console Application)

📌 Project Description

This project is a console-based Brick Breaker Game developed in Java using Low-Level Design (LLD) principles.
The game simulates a ball hitting bricks on a board. When the ball hits a brick, the brick's strength decreases and may get destroyed.

The player controls the direction of the ball using commands.

🎯 Features

7 × 7 Game Board

Ball movement in three directions:

ST → Straight

LT → Left

RT → Right

Brick collision detection

Brick destruction after strength becomes zero

Ball life reduction on hitting bricks

Console-based board display

Modular design using packages
